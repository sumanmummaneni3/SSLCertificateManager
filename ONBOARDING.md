### Skills to Know About
- `/compact` — summarizes the conversation so far and compresses context, freeing up room for longer sessions without losing thread. Use it when you're deep in a debugging session and hitting the context limit.
- `/clear` — wipes the conversation and starts fresh. Good when switching tasks completely.
- `/memory` — reads and updates Claude's persistent memory for this project. Check it if Claude seems to have forgotten something from a prior session.
- `/agents` — shows available specialized agents (architect, backend-engineer, frontend-engineer, etc.). Useful when you want to delegate a focused task.
